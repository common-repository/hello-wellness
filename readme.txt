A plugin inspired by Hello Dolly that displays fun quotes about wellness at the top of your admin screen.

== Description ==

Built using the base code of the popular plugin, Hello Dolly, this plugin gives
users a regular reminder to consider their physical and mental wellness.
Thank you to Matt Mullenweg for creating Hello Dolly.

== License ==

Released under the terms of the GNU General Public License.

== Installation ==

1. Upload the zipped plugin files to the `/wp-content/plugins/hello-wellness` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= What do you hope to achieve with this plugin? =

I wanted to make it easy to remind others of the need to stop for a moment and consider their wellness in the spirit of #wpmom.

= Can I suggest additional quotes to be added to future versions? =

Of course! Submit any ideas you have to me via Twitter at @rkoffy.
